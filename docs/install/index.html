Developer install

  1. Run npm install and npm run build.
  2. Open chrome://extensions, enable Developer Mode, and load apps/extension/dist.
  3. Build the native host and register com.describeops.native with the helper script for your platform.
  4. Open the extension side panel and use Check companion to confirm the local bridge.
DescribeOps processes only content you own, license, or are authorized to make accessible. It does not bypass DRM, paywalls, or private platform restrictions.
