# Three-Minute Demo Script

Target runtime: 2 minutes 45 seconds.

## 0:00-0:20 - Problem

Open a training page with an embedded video and no audio-description track. Explain that many videos live inside LMS, intranet, or private portal pages where a URL-only generator is not enough.

## 0:20-0:45 - Installable Browser Agent

Click the DescribeOps extension. The side panel opens on Detect. It reports detected video, headings, visible text, captions, and one canvas or embedded-player limitation when present.

## 0:45-1:15 - Edge Companion

Open the native companion health card. Show version, storage path, SQLite cache, FFmpeg status, and weak-network queue support. Mention that the browser never stores Qwen credentials.

## 1:15-1:55 - Qwen Agent Workflow

Create an authorized analysis job. Show planned evidence extraction: transcript windows, sampled frames, OCR, speech gaps, and page context. The Qwen Agent Society drafts audio-description cues and QA flags a risky unsupported claim.

## 1:55-2:25 - Human Review and Memory

Edit the flagged cue. Save the wording preference. Show that future jobs can retrieve the saved organization style rule while job-specific facts expire or stay scoped.

## 2:25-2:45 - Playback and Offline Value

Play the video with synchronized audio-description cues. Switch to offline package mode and show cached playback/review assets remain available.

## Close

DescribeOps turns Qwen Cloud into a production-minded accessibility operations agent: browser-installed, multimodal, memory-aware, human-reviewable, measurable, and useful for real schools, employers, and creators.
